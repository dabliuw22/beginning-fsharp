# beginning-fsharp

## Create Project:
```bash
$ dotnet new console -lang F# -o src/App
```
## Dependencies:
```bash
$ dotnet paket add FSharp.Control.Reactive --version 5.0.2 --project App
```

## Install:
```bash
$ dotnet paket install
```

## Run Project:
```bash
$ dotnet run --project src/App
```
